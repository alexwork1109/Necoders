## Выбранный кейс

> **Интеллектуальный отбор данных: Конструктор аналитических выборок для бюджетного планирования**

Проект помогает формировать аналитические выборки из нескольких источников данных: бюджетной росписи, соглашений, госзаказа и выгрузок БУАУ. Пользователь выбирает объект, показатели и период, a система собирает результат в таблицу и графики без ручной склейки файлов.

## Что делает решение

- загружает структурированные CSV-файлы из разных источников;
- связывает данные через бюджетные коды и коды объектов;
- позволяет искать объект по коду или названию;
- показывает план, кассу, контракты, обязательства и платежи;
- сравнивает показатели по датам и периодам;
- экспортирует результат в Excel.

## Развёртывание и запуск

### Локально для разработки (Windows)

Наиболее быстрый способ запуска среды разработки при помощи подготовленных скриптов:

```powershell
prepare.cmd
dev.cmd
```

*Frontend будет доступен на `http://localhost:5173`, API - на `http://localhost:5000`.*

### Локально для разработки (Linux / РЕД ОС)

```bash
chmod +x ./prepare.sh ./dev.sh
./prepare.sh
./dev.sh
```

### Запуск через Docker (РЕД ОС / Прод-сборка)

Проект полностью упакован в единый Docker-контейнер и готов к развертыванию:

```bash
docker build -t constructor2026 .
docker run -p 8080:8080 -v $(pwd)/data:/data constructor2026
```

После запуска приложение будет доступно по адресу `http://localhost:8080`.
